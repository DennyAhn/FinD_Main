# MCP 서비스 커버 범위 및 예시 질문

## 📊 지원 도구 목록 (13개)

### 1. 회사 검색 및 프로필
- **도구**: `search_company_by_name`, `fetch_company_profile`
- **기능**: 회사명(한글/영문)으로 티커 검색, 회사 프로필 정보 조회

### 2. 재무 비율 (PER, PBR, ROE, ROA 등)
- **도구**: `fetch_company_key_metrics`
- **기능**: 연간/분기별 주요 재무 비율 조회 및 인사이트 제공

### 3. 손익계산서
- **도구**: `fetch_company_income_statements`
- **기능**: 연간/분기별 손익계산서 데이터 조회

### 4. 대차대조표
- **도구**: `fetch_company_balance_sheets`
- **기능**: 연간/분기별 대차대조표 데이터 조회

### 5. 현금흐름표
- **도구**: `fetch_company_cash_flows`
- **기능**: 연간/분기별 현금흐름표 데이터 조회 및 건강성 분석

### 6. 주가 정보
- **도구**: `fetch_stock_quote`, `fetch_market_time_series`
- **기능**: 현재 시세, OHLCV 시계열 데이터 조회

### 7. 뉴스
- **도구**: `search_summarized_news`
- **기능**: 특정 종목 관련 요약 뉴스 조회

### 8. 애널리스트 평가
- **도구**: `fetch_analyst_ratings`
- **기능**: 투자의견, 목표주가 조회

### 9. 실적 발표
- **도구**: `fetch_earnings_calendar`
- **기능**: 과거 실적 발표 기록 및 EPS/매출 실적 조회

### 10. 내부자 거래
- **도구**: `fetch_insider_trades`
- **기능**: 내부자 거래 내역 조회

### 11. 대화 이력
- **도구**: `get_chat_history`
- **기능**: 사용자 대화 이력 조회

## 🎯 답변 방식 특징

### 1. **도구 기반 답변**
- 모든 답변은 도구에서 가져온 실제 데이터 기반
- 추측이나 일반 지식으로 답변하지 않음

### 2. **캐시 전략**
- 재무제표: `report_date` 기준 3개월 캐시
- 프로필: 30일 캐시
- 기타: API별 캐시 정책 적용

### 3. **인사이트 제공**
- 단순 데이터 나열이 아닌 분석 및 해석 제공
- PER/PBR: 평균 대비 상대적 위치 분석
- 현금흐름: 건강성 점수 및 등급 제공
- 추이 분석: 이전 기간 대비 변화율 제공

### 4. **한국어 답변**
- 모든 최종 답변은 한국어로 제공
- 전문 용어는 설명과 함께 제공

## 📝 사용 팁

1. **티커를 모를 때**: 회사명으로 먼저 검색
   - "애플" → `search_company_by_name` → "AAPL" 확인

2. **분기별 데이터 요청 시**: 명확히 "분기별" 또는 "quarter" 언급
   - "올해 분기별 PER" → `period="quarter"` 사용

3. **복합 질문**: AI가 자동으로 여러 도구를 조합하여 답변
   - "애플 주가 하락 이유" → 뉴스 + 실적 + 평가 조합

4. **데이터 없을 때**: 명확히 "데이터를 찾을 수 없습니다"라고 안내
